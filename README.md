# DentalClinic Management System 🦷

A comprehensive Management System for a Dental Clinic, developed as part of my Software Engineering studies. The project focuses on backend architecture, database management, and clean code principles.

## 🚀 Key Features
- **Patient Management:** CRUD operations for managing patient records and medical history.
- **Appointment Scheduling:** Efficient system for booking and managing dental appointments.
- **Treatment Tracking:** Keeping track of performed treatments and future plans.
- **Clinic Dashboard:** Overview of clinic activities and statistics.

## 🛠️ Tech Stack
- **Language:** C#
- **Framework:** .NET Core / Web API
- **Database:** SQL Server (using Entity Framework Core)
- **Documentation:** Swagger UI (for API testing)
- **Version Control:** Git & GitHub

## 🏗️ Architecture
The project is built using the **Repository Pattern** to ensure a clean separation of concerns between the data access layer and the business logic. 

- **Controllers:** Handle HTTP requests and responses.
- **Services/Repositories:** Manage data logic and interaction with the database.
- **Models:** Define the data structures for Patients, Appointments, and Treatments.
